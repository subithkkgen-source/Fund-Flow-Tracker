⚙ Supabase Not Configured

You need to add your Supabase project URL and API key to the app before it can connect to the shared database. Open index.html in a text editor and replace these two lines near the top: 

const SUPABASE_URL  = 'YOUR_SUPABASE_URL';
const SUPABASE_ANON = 'YOUR_SUPABASE_ANON_KEY';

Get these values from your Supabase project → Settings → API.
See SETUP-GUIDE.html for step-by-step instructions.
